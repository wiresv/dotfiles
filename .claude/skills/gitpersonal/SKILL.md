---
name: gitpersonal
description: Set the global git user.name and user.email to Austin's personal identity.
---

Set the global git identity for this user by running:

```
git config --global user.name "wiresv"
git config --global user.email "aaustinhwx@gmail.com"
```

Then confirm by running:

```
git config --global --get user.name
git config --global --get user.email
```

Report the resulting values back to the user.
